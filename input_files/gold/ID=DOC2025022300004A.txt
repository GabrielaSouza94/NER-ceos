Descrição Geral
O documento apresenta um mapeamento completo das ferramentas e sistemas utilizados em cada macroprocesso das principais áreas da Globo. Ele detalha quais soluções tecnológicas são aplicadas para planejamento, alocação de recursos, talentos artísticos, RH, esportes, jornalismo e integrações tecnológicas.

A estrutura do documento segue uma abordagem hierárquica, onde cada macroprocesso tem seus sistemas específicos listados, indicando quais plataformas são utilizadas para planejamento, execução, monitoramento e integração de dados.

Resumo do Conteúdo
O documento apresenta um levantamento detalhado das tecnologias utilizadas pela Globo para suporte aos macroprocessos operacionais. Cada área possui um conjunto de ferramentas específico, muitas vezes com soluções legadas ou customizadas.

Os sistemas utilizados variam de planilhas Excel a ferramentas avançadas como SAP BRIM, Oracle HCM e Camunda BPM, demonstrando a complexidade da infraestrutura tecnológica da Globo.

A integração entre sistemas é um ponto crítico, com a Decoupling Layer desempenhando um papel fundamental na comunicação entre ferramentas. Ferramentas como Grafana e Prometheus são usadas para monitoramento e suporte, enquanto BigQuery, Data Lake e BI auxiliam na análise de dados.

Principais Insights

1️ A Globo depende de múltiplas ferramentas para diferentes processos:
MediaPulse, SAP BRIM e Oracle EBS são fundamentais para planejamento e gestão financeira.
Oracle HCM e FPW são cruciais para gestão de talentos e RH.
A diversidade de ferramentas gera desafios de integração e complexidade operacional.

2️ Ainda há forte dependência de processos manuais:
Planilhas Excel e e-mails são amplamente utilizados, principalmente na transição entre longo e curto prazo e na gestão de talentos esportivos.
Isso gera riscos operacionais, atrasos e dificuldades na atualização de informações.

3️ A integração de sistemas precisa ser robusta:
A Decoupling Layer desempenha um papel central na comunicação entre sistemas distintos.
APIs customizadas são utilizadas para conectar MediaPulse, SAP BRIM, FPW e Oracle HCM.
Monitoramento ativo (Grafana, Prometheus) é essencial para evitar falhas e gargalos.

4️ A área de Talentos Artísticos possui um modelo tecnológico fragmentado:
O processo de gestão de talentos no Esporte ainda é manual, sem um sistema estruturado.
Diferentes ferramentas são usadas para captação, planejamento, contratação e pagamento, mas não há um sistema único que integre essas funções.

5️ Há espaço para automação e otimização:
Camunda BPM e automação via filas e brokers já são utilizados, mas podem ser expandidos para reduzir tarefas manuais.
A implementação de um sistema unificado para talentos e esportes poderia melhorar a eficiência.
